# BebeParceiro
O projeto Bebê Parceiro é natural de Maringá/PR. Seu foco é ajudar gestantes ou mães com filhos de até 1 ano de idade que não possuem recursos econômicos para comprar as roupinhas e/ou acessórios de higiene e etc...

Com o crescimento do projeto, as **planilhas de gerenciamento começaram a ficar desorganizadas** e **o trabalho tornou-se cada vez mais demorado e desgastante**.

Outro problema que surge é o de **proteger os dados dos beneficiados**. Várias informações sobre endereço, contato e sobre a pessoa em geral são armazenadas nesse processo e com o estado inicial do projeto, **todos os voluntários tinham acesso à elas**.

Tendo em vista toda a problemática, está sendo desenvolvido um sistema web para o projeto. As tecnologias usadas são **Laravel 8** e **Vue.js 3**.

Para o Back-End, pretende-se usar os princípios de **Código limpo** (o dsenvolvedor ainda esetá em processo de aprendizado) e *Domain Driven Development* para que futuras manutenções e adições de funcionalidades - caso venham a ser necessárias - não sejam penosas.
